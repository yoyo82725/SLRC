# SLRC
學資中心網站

2012/08 專案，大一時製作，約20個動態頁面可供管理者編輯，含公告、活動花絮、討論版、表單下載等。
<ul>
<li>無刷新(Refresh)為特色</li>
<li>防範sql injection攻擊</li>
<li>原生Html、Css、JQuery、PHP製成</li>
<li>可供後台編輯功能20處</li>
</ul>
